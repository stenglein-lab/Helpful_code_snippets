This directory contains files I used to perform a batch submission of several dozen virus genome sequences to Genbank.

I did this using the NCBI [table2asn](https://www.ncbi.nlm.nih.gov/genbank/table2asn/) utility.  

More details about the input files and workflow are provided in the [create_files_for_genbank_submission](./create_files_for_genbank_submission) script.
